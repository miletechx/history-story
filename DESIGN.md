# DESIGN.md

## 气质与意象
- **关键词**：极简、优雅、故事感、沉浸聆听
- **场景描述**：苹果产品发布会般的简洁优雅，大量留白，高质量视觉，让用户感受到这是一个精心设计的产品。历史故事在温暖的光影中娓娓道来。

## 视觉策略
- **整体风格**：苹果风格极简设计，大量留白，精致的排版
- **主色调**：中性色调为主，琥珀金色作为品牌色点缀
- **图形语言**：简洁的几何形状、柔和的圆角、精致的阴影

## 配色方案
| 用途 | 颜色 | 说明 |
|------|------|------|
| 主背景 | #fafafa / 纯白 | 苹果风格大量留白 |
| 品牌色 | #d4a373 / 琥珀金 | Logo和强调元素 |
| 主文字 | #1d1d1f / 深灰黑 | 苹果标准文字色 |
| 次文字 | #86868b / 中灰 | 辅助说明文字 |
| 强调色 | #d4a373 / 琥珀金 | 按钮、链接高亮 |
| 分割线 | #d2d2d7 / 浅灰 | 卡片边框、分割线 |

## 字体排版
- **标题字体**：SF Pro Display / 苹方-简 中粗体（苹果风格）
- **正文字体**：SF Pro Text / 苹方-简 常规体
- **排版节奏**：超大标题（64px+）、宽松行距（1.5）、舒适的段落间距

## 设计原则（苹果风格）
1. **极简主义**：去除一切不必要的装饰，聚焦内容
2. **大量留白**：让内容有呼吸空间，提升高级感
3. **高质量图片**：使用精美的摄影和插画，避免低质量素材
4. **精致细节**：圆角、阴影、过渡动画都要细腻
5. **一致性**：所有元素保持统一的设计语言
6. **可访问性**：确保足够的对比度和可读性

## 动效与交互
- **运动性格**：流畅、优雅、不急躁
- **过渡效果**：ease-out缓动，300ms标准时长
- **滚动效果**：视差滚动、渐入动画
- **悬停效果**：轻微放大、颜色变化

## 页面结构

### 首页（苹果风格）
- **Hero区域**：全屏背景图 + 产品名称 + 核心价值主张 + CTA按钮
- **功能介绍**：三列卡片，展示核心功能
- **年龄适配**：展示不同年龄段的故事风格差异
- **示例展示**：展示真实的故事内容和配图
- **用户评价**：（MVP阶段可省略）
- **CTA区域**：引导用户开始体验

### 播放页
- **极简播放器**：故事标题 + 流式文字 + 简洁播放控件
- **进度反馈**：优雅的加载动画
- **配图展示**：分镜配图以画廊形式展示

## 设计禁忌
- ❌ 不要使用过多的装饰元素
- ❌ 不要使用低质量的图片素材
- ❌ 不要使用过多颜色，保持克制
- ❌ 不要使用过小的字号（最小14px）
- ❌ 不要使用刺眼的高对比度配色
- ❌ 不要在大量留白区域放置干扰元素
